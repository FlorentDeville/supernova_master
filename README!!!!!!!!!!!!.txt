Do not compile the Supernova project, it won't work. You need to compile the Demo1 solution in /sample/Demo1.

You can use a X360 controller:
 - left stick : up/down left/right
 - right stick : turn camera
 - right trigger : move forward
 - left trigger : move backward
 - X : launch box
 - Y : hide/show watch
 - A : show/hide collision points (only for non sleeping objects)
 - B : wireframe/solid
